# Hi, I'm Daniel

I'm an aspiring game developer and web design enthusiast currently studying Computer Science at BYU–Idaho. I’m interested in how games function at a systems level, including scripting, UI logic, gameplay mechanics, and engine architecture.

I am currently developing **Arcane Chronicles**, a Godot-based game that allows players to mix and combine spell attributes to create unique spells. The spell‑crafting system already supports an extremely large number of possible combinations, and I continue to expand it as I learn more about GDScript and the Godot Engine.

I also enjoy front-end web development and have built several small projects that help me practice JavaScript, UI behavior, and clean interface design. I am now working toward learning backend development to build more complete and data‑driven applications.
